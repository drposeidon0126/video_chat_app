# Peek

## Local setup

```sh
npm install
```

Run server and client
```sh
npm run up
```

Run server only
```sh
npm run server
```

Run client only
```sh
npm run client
```

Run tests
```sh
npm run test
```

Test driven development
```sh
npm run tdd
```

GitHub Action PR Tasks simulation
```sh
./scripts/deploy.local.sh
```

Build
```sh
npm run affected:build -- --prod
```

Format files with prettier
```sh
npm run format
```
